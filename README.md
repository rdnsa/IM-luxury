# I'M LUXURY BAG

Modern luxury fashion showcase website built with Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

## Tech

- Next.js (App Router)
- TypeScript (strict)
- Tailwind CSS
- Framer Motion
- Clean Architecture (`domain`, `application`, `infrastructure`, `presentation`)
